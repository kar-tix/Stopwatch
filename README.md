# Stopwatch

A simple stopwatch made using technology: HTML, CSS and JavaScript. </br>
This stopwatch represents the time in HH:MM:SS:MsMsMs, where "Ms" is milliseconds. </br>
The clock has functions **start**, **stop** and **reset**. <br>
